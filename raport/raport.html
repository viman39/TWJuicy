Introducere

    Juicy este o aplicatie web care pune la dispozitia utilizatorilor interpretarea a doua roluri:
  • Vanzator
  • Cumparator
Vanzatorul:
  • adauga produse noi care pot fi cumparate
  • vizualizeaza evolutia vanzarilor si a stocului in timp real
  • sterge produsele pe care nu le mai vinde
Cumparatorul:
  • creeaza liste de cumparaturi
  • controleaza lista de cumparaturri cat si cantitatea cumparata
  • finalizeaza comenzi
Alte intrebuintari ale site-ului:
  • trimite email de confiramre
    • cand o persoana noua isi face cont(vanzator/cumparator)
    • cand un produs este vandut(vanzatorului)
    • cand o camanda este finalizata(cumparatorului)
  • adaugarea produselor direct din shop

Structura

Pagina principala. Cuprinde un meniu cu 3 butoane:
  • Home: butonul de acasa cu informatii despre site
  • Contact: care redirectioneaza la un chestionar pentru a deveni vanzator
  • Login: care deschide o fereastra cu un formular pentru logare
    • care poate duce mai departe la crearea unui cont de cumparator
Odata logat sunt doua flow-uri.
  • In cazul in care este un client apar butoanele
    • Shop: care prezinta produsele
    • Shopping history: neimplementat inca
    • Logout: pentru terminarea sesiunii de cumparaturi
    • Cart: pentru vizualizarea liste de cumparaturi
      • care mai departe duce la finalizarea comenzii prin butonul de checkout
  • In cazul in care este un vanzator apar butoanele
    • Shop: care prezinta produsele
    • Produsele tale: care prezinta produsele pe care le detii tu ca vanzator
      • care mai departe poate duce la Adauga produs pentru a adauga produse noi
    • Logout: pentru terminarea sesiunii de cumparaturi
    • Cart: pentru vizualizarea liste de cumparaturi
      • care mai departe duce la finalizarea comenzii prin butonul de checkout

Tehnologii utilizate

HTML5(HyperText Markup Language), o tehnologie nucleu pentru internet, propusa intial pentru browser-ul Opera,
este un limbaj pentru structurarea si prezentarea continutului WWW(World Wide Web)
CSS(Cascading Style Sheets), este un standard pentru formatarea elementelor unui document HTML.
PHP(Php: Hypertext Preprocessor), este un limbaj de programare folosit initial pentru a produce pagini web dinamice,
este folosit in scara larga in dezvoltarea paginilor si aplicatiilor web.
MySQLi este un sistem de gestiune a bazelor de date rela»õionale.

Baza de date

Baza de date este alcatuita din 7 tabele bine structurate care permit usoara gestionare a informatiilor cat si adaugarea de coloane/tabele noi cu usurinta.
  • clienti: id_client, adresa, email, nume, prenume, parola(incriptata md5)
  • vanzator: id_vanzator, email, parola(incriptata md5)
  • produse: id_produs, id_vanzator, nume, pret, acidulat, arome, path_poza(calea catre poza din shop), nou(indice care spune daca e adaugat in ultima luna), data_adaugarii
  • detine: id_vanzator, id_produs, cantitate
  • lista_cumparaturi: id_lista_cumparaturi
  • plateste_pentru: id_client, id_lista_cumparaturi, finalizare, vanzator(coloana care ne spune daca lista e realizata de pe contul unui vanzator)
  • cantitate_cumparata: id_lista_cumparaturi, id_produs, cantitate
